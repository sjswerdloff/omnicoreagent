# OmniRex Due Diligence System - DeepAgent Edition

An AI-powered due diligence system specifically designed for African SME evaluation. Powered by OmniCoreAgent DeepAgent technology.

## 🚀 What It Does

This system replaces the manual, sequential due diligence process with a massively parallel AI workforce.

1.  **Lead Agent** receives a target company name.
2.  **Context Analysis**: It understands the African market context (FX risk, regulatory fragmentation, infrastructure challenges).
3.  **Parallel Subagents**: It spawns 6+ specialist agents simultaneously:
    *   **Financial Analyst**: Revenue, unit economics, projections.
    *   **Market Researcher**: TAM/SAM/SOM, competition.
    *   **Team Assessor**: Founder background, execution track record.
    *   **Risk Evaluator**: Macro risks, regulatory compliance.
    *   **Impact Specialist**: ESG alignment, job creation.
4.  **Synthesis**: It consolidates findings into investor-ready outputs.

**Result**: A comprehensive due diligence packet in < 10 minutes (vs 60+ mins manually).

## 🛠️ Prerequisites

You must set the following environment variables:

```bash
# Required for internet research
export TAVILY_API_KEY=your_key_here

# Required for the LLM
export LLM_API_KEY=your_key_here
```

## 📦 Installation

```bash
pip install -r requirements.txt
```

## 💻 Usage (Pure Terminal)

Run the evaluation directly from your terminal. No TUI required.

```bash
# Navigate to the cookbook directory
cd cookbook/deep_agent/deep_ai_due_diligence

# Run the runner directly
python engine/deep_agent_runner.py "Company Name"
```

**Example:**

```bash
python engine/deep_agent_runner.py "Paystack"
```

## 📊 Outputs Generated

The system automatically generates three key documents in the `outputs/` directory:

1.  **Infographic Dashboard** (`.png`): A Goldman Sachs-style visual summary with key metrics, risk score gauge, and top highlights.
2.  **Investment Report** (`.html`): A McKinsey-style detailed report with executive summary, financial tables, and deep analysis.
3.  **Decision Memo** (`.md`): A structured internal memo with the final FUND/PASS recommendation.

## 🏗️ Architecture

- **DeepAgent**: The orchestration core that manages subagents.
- **Dynamic Spawning**: Subagents are created on-demand based on the investigation needs.
- **Tool Use**: Agents have access to real-time web search, financial charting, and report generation tools.
