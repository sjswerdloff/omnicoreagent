# How to Use Vula TUI - Quick Guide

## 🚀 Launch

```bash
uv run cookbook/deep_agent/vula_due_diligence/main.py
```

## 📝 Step-by-Step Instructions

### Dashboard Screen:
1. You'll see 3 buttons:
   - **[1] 🔍 Evaluate Single Company**
   - **[2] 📊 Batch Evaluate (CSV)**
   - **[3] 📁 View Recent**

### To Evaluate a Company:

**Step 1:** Press **[1]** on keyboard

**Step 2:** You'll see:
```
🏢 Company Name:
👉 Click on the box below or press TAB, then type company name
[Click here and type company name...]
[🚀 Start Evaluation]
```

**Step 3:** **CLICK on the input box** (the one that says "Click here...")
   - OR press **TAB** key to focus it

**Step 4:** Type the company name (e.g., "Acme AgroTech")

**Step 5:** Press **ENTER** or click **[🚀 Start Evaluation]** button

**Step 6:** Watch DeepAgent work! You'll see:
   - Research phase progress
   - Subagents working in parallel
   - Live log updates
   - Final results with confidence scores

## ⌨️ Important Keys:

- **TAB** - Move focus between input/button
- **ENTER** - Submit (when input is focused)
- **MOUSE CLICK** - Click on input box to type
- **ESC** - Go back
- **Q** - Quit application

## 💡 Pro Tips:

1. **Always CLICK or TAB first** before typing
2. Input field has a **cyan border** when focused
3. You can use **arrow keys** and **TAB** to navigate
4. Press **H** for help anytime

## 🐛 Troubleshooting:

**Problem:** "I can't type!"  
**Solution:** Click on the input box or press TAB first

**Problem:** "Nothing happens when I press Enter"  
**Solution:** Make sure input box is focused (has cursor blinking inside)

**Problem:** "Button doesn't work"  
**Solution:** Type a company name first, then click button
